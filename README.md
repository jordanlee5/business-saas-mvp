# 业务数据管理 SaaS MVP

这是一个用于业务数据上传、承接、凭证识别、匹配审核、经营统计与结算查看的 SaaS MVP 项目。

当前版本定位为 **MVP 阶段的稳定开发版**，主要用于本地演示、业务流程验证和后续功能迭代。

---

## 当前版本

**v0.1.0 - SaaS MVP 初版稳定版本**

当前版本已经完成从上传方上传业务数据，到管理员承接、凭证上传识别、匹配审核、业务状态回写、经营统计与上传方结算查看的核心闭环。

---

## 核心功能

### 1. 账号与权限

- 管理员账号登录
- 上传方账号登录
- 管理员与上传方数据权限隔离
- 管理员可创建和维护上传方账号
- 上传方只能查看与自己相关的数据

### 2. 业务数据上传与承接

- 上传方上传 Excel 服务需求
- 管理员查看上传批次
- 管理员可对上传批次进行承接或拒绝
- 仅“已承接”的批次会进入后续业务数据、统计、导出和凭证匹配链路
- “待承接 / 已拒绝”的批次不会进入经营核算和凭证匹配来源

### 3. 业务数据管理

- 管理员查看全部业务数据
- 上传方查看自己的业务数据
- 支持按上传方、关键词、日期、审核状态、承接状态筛选
- 支持业务数据分页
- 支持业务数据导出 Excel
- 支持进入单条业务详情页
- 业务数据列表展示：
  - 最新审核状态
  - 最新匹配状态
  - 已通过凭证金额
  - 剩余金额
  - 导入时间

### 4. 业务详情页

- 展示单条业务的基础信息
- 展示业务金额、已通过凭证金额、剩余金额、付款状态
- 展示关联凭证审核记录
- 管理员可看到完整审核链路
- 上传方侧仅展示对验收有价值的已通过凭证
- 当业务已足额付款时，系统自动将业务状态视为已通过

### 5. 凭证上传、OCR 识别与匹配

- 支持上传 PNG / JPG / JPEG 图片凭证
- 系统识别凭证文字
- 提取或识别凭证金额
- 根据姓名、银行卡号、金额等信息与业务数据进行匹配
- 支持选择指定已承接业务批次进行匹配
- 支持凭证哈希去重
- 支持重复凭证复用匹配，避免因误选批次导致凭证无法再次使用

### 6. 匹配结果审核

- 管理员查看待审核、已通过、已驳回的匹配记录
- 支持按上传方、姓名、审核状态筛选
- 支持分页
- 支持单条审核通过 / 驳回
- 支持批量审核
- 支持从业务详情页精准跳转到指定审核记录
- 支持审核完成后返回业务数据管理页继续处理

### 7. 经营数据与结算

- 管理员查看经营数据看板
- 管理员导出经营数据
- 上传方查看自己的结算报表
- 上传方导出自己的结算数据
- 系统根据下游服务费率、上游成本费率计算经营数据

### 8. 页面与交互

- 统一侧边栏导航
- 管理员侧与上传方侧菜单区分
- 统一紧凑化页面风格
- 多页面分页样式统一
- 业务流程入口清晰化
